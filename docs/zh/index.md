# 欢迎使用 BERT-CLAM 框架

**BERT-CLAM** 是一个专为**持续学习 (Continual Learning)** 研究而设计的、灵活且可扩展的实验框架。

---

### **项目愿景**

在持续学习领域，研究者们常常需要花费大量时间搭建实验环境、实现基线模型，并确保结果的可复现性。`BERT-CLAM` 旨在通过提供一个模块化、易于扩展的平台，将研究者从这些重复性工作中解放出来，让他们能够更专注于算法和策略本身的创新。

我们相信，一个好的研究框架应该具备以下特点：

-   **模块化设计**: 轻松替换或添加新的持续学习策略 (如 EWC, Memory Replay 等)。
-   **易于配置**: 通过简单的 JSON 文件即可定义复杂的实验流程。
-   **结果可复现**: 提供标准化的评估流程和工具，确保实验结果的一致性。
-   **社区驱动**: 欢迎来自社区的贡献，共同构建一个更强大的工具集。

### **快速开始**

想要立即开始探索吗？请跟随我们的 **快速入门指南**，在几分钟内运行你的第一个持续学习实验。
